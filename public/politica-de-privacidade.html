Política de Privacidade

Última atualização: Janeiro de 2026

Resumo: Coletamos apenas os dados necessários para atendê-lo melhor. Seus dados são protegidos e usados exclusivamente para orçamentos e contato comercial.

1. Quais Dados Coletamos

Dados Pessoais Fornecidos por Você:

  • Nome: Para personalizar o atendimento
  • Cidade: Para verificar nossa área de atendimento
  • Bairro/Região: Para calcular deslocamento e agendar visitas
  • Tipo de serviço: Para preparar orçamento adequado
  • Número de WhatsApp: Para contato direto (quando fornecido)

Dados Coletados Automaticamente:

  • Endereço IP: Para segurança e estatísticas
  • Tipo de dispositivo: Para melhorar a experiência do site
  • Navegador utilizado: Para compatibilidade
  • Páginas visitadas: Para entender o interesse dos visitantes
  • Horário de acesso: Para estatísticas de uso

2. Como Coletamos os Dados

  • Formulários do site: Quando você solicita orçamento
  • Botões de WhatsApp: Ao clicar nos botões de contato
  • Cookies: Para melhorar sua experiência de navegação
  • Redes sociais: Quando acessa nossos perfis (Instagram/Facebook)

3. Para Que Usamos Seus Dados

Finalidades Principais:

  • Atendimento: Responder suas dúvidas e solicitações
  • Orçamentos: Calcular preços e preparar propostas
  • Agendamento: Marcar visitas técnicas
  • Contato comercial: Enviar informações sobre nossos serviços

Finalidades Secundárias:

  • Melhoria do site: Analisar como você usa nosso site
  • Segurança: Prevenir fraudes e uso inadequado
  • Estatísticas: Entender preferências dos clientes

4. Compartilhamento de Dados

Compartilhamos seus dados apenas com:

  • WhatsApp (Meta): Quando você usa nossos botões de contato
  • Instagram (Meta): Se você acessa nosso perfil
  • Facebook (Meta): Se você acessa nossa página
  • Google: Para análise de tráfego do site (se aplicável)

Importante: Nunca vendemos, alugamos ou cedemos seus dados para terceiros não relacionados aos nossos serviços.

5. Armazenamento e Segurança

  • Seus dados são armazenados em servidores seguros
  • Usamos criptografia para proteger informações sensíveis
  • Acesso restrito apenas a funcionários autorizados
  • Backup regular para evitar perda de dados
  • Dados de orçamento são mantidos por 2 anos para histórico

6. Seus Direitos (LGPD)

Você tem direito a:

  • Saber quais dados temos sobre você
  • Corrigir informações incorretas
  • Excluir seus dados (quando possível)
  • Limitar o uso dos seus dados
  • Receber uma cópia dos seus dados
  • Revogar seu consentimento a qualquer momento

7. Cookies

Usamos cookies para:

  • Lembrar suas preferências
  • Melhorar a velocidade do site
  • Analisar como você usa nosso site
  • Personalizar sua experiência

Você pode desabilitar cookies no seu navegador, mas isso pode afetar algumas funcionalidades do site.

8. Menores de Idade

Não coletamos intencionalmente dados de menores de 18 anos. Se você é menor de idade, peça para um responsável entrar em contato conosco.

9. Alterações desta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quando isso acontecer, avisaremos através do site e por e-mail (se tivermos seu contato).

10. Consentimento

Ao usar nosso site e fornecer seus dados, você consente com esta Política de Privacidade. Você pode retirar seu consentimento a qualquer momento entrando em contato conosco.
